# 项目需求说明

## 1. 项目定位
- 跨平台 Electron 桌面记账软件
- 主要面向 Windows 用户，兼容 macOS / Linux
- 界面简洁直观，主色调为淡蓝色
- 本地数据存储，支持 Excel 导入/导出
- ~~支持多用户登录、密码保护和自动备份~~（第一版为单用户模式，多用户功能延后）

## 2. 核心功能

### 2.1 记账管理
- 添加收支记录：金额、类型、类别、子类别、日期、备注
- 编辑记录、删除记录（支持批量删除）
- 记录列表按时间倒序展示
- 支持按日期/类型/分类筛选
- 支持快速复制上一条记录，时间自动设为当前日期
- 支持撤销/重做操作：记账记录、分类变更、预算修改

### 2.2 分类管理
- 预设常用分类
- 用户可增、删、改分类
- 删除分类前检查是否有关联记录
- 支持图标自定义、颜色设置
- 支持拖拽排序一级分类
- 支持二级子分类树状结构

### 2.3 数据统计
- 月度收支总览：收入、支出、结余
- 支出类别占比饼图
- 近 7 天 / 30 天趋势折线图
- 年度对比柱状图
- 基于历史平均的收支预测
- 自定义统计时间范围

### 2.4 预算管理
- 设置月度总预算，显示剩余额度
- 分类独立预算
- 支持季度预算和年度预算
- 实时超预算弹窗提醒
- 预算剩余比例警告

### 2.5 用户与安全
- 本地账户登录与密码保护
- 多用户数据隔离与切换
- 自动备份与恢复
- 支持备份文件导入恢复

## 3. 数据导入 / 导出
- Excel `.xlsx` 格式导入导出
- CSV 格式作为补充支持
- 导入时字段映射与异常提示

## 4. 质量标准
- UI 简洁，可读性强
- 关键操作可撤销/重做
- 数据变更需持久化到本地数据库
- 分类删除与预算修改需校验一致性
- 导入导出需稳定处理异常数据
