# 设计规范

## 1. 视觉风格
- 主色调：淡蓝色
- 辅助色：浅灰、白色、深蓝
- 使用清晰的卡片、分区布局
- 统一圆角按钮与表格单元格边角
- 保持充足留白，减少视觉负担

## 2. 组件规范
- 按钮：主操作使用蓝色背景，次要操作使用灰色边框
- 表单：标签靠左，输入控件对齐一致，支持快捷键与回车提交
- 卡片：标题 + 内容区 + 操作按钮，信息分层清晰
- 图表：配色柔和、数据标签可读，支持悬浮提示

## 3. 页面布局
- 左侧导航栏：固定、图标+文字
- 顶部栏：操作快捷按钮、搜索/筛选入口
- 内容区：分为统计区、列表区、表单区
- 弹窗与侧滑层：用于新增/编辑、导入导出、预算操作

## 4. 交互规范
- 重要变更前提示确认（删除分类、删除记录、覆盖数据）
- 表单错误采用红色提示并聚焦第一个错误项
- 引导用户使用“快速复制上一条”时，显示简短说明
- 预算超标时弹窗提醒 + 仪表板红色警示

## 5. 用户体验
- 数据加载要有占位符/加载指示
- 列表分页或无限滚动保证性能
- 支持表格行高亮、可展开子分类
- 导入导出流程应分步展示进度与结果

## 6. 色彩与字号建议
- 主要文本：#1f2d3d
- 次要文本：#5a6b7c
- 背景：#f7fbff / #ffffff
- 主按钮：#4a90e2
- 警告色：#e65a5a

## 7. 图标与分类展示
- 分类图标应简洁易识别
- 支持图标与颜色组合展示分类层级
- 二级分类缩进显示，选择时可展开/折叠
