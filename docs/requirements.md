# 项目需求说明

## 1. 项目定位
- 跨平台个人记账软件
- 桌面端：Electron + React（Windows）
- 移动端：PWA（iOS Safari 添加到主屏幕）
- 完全离线：sql.js + localStorage，无后端依赖
- 界面风格：极简现代，参照 Refero 设计规范，移动优先

## 2. 核心功能

### 2.1 记账管理 ✅
- 添加收支记录：金额、类型、分类、子分类、日期、备注
- 编辑/删除记录
- 按时间倒序列表展示
- 按类型/日期/关键词筛选
- 快速复制上一条记录
- 撤销/重做操作

### 2.2 分类管理 ✅
- 20 个预设分类（13 支出 + 7 收入）
- 8 个预设子分类（餐饮→正餐/零食 等）
- 用户增删改分类 + 自定义图标（Emoji）
- 删除前检查关联记录
- 分类排序（↑↓ 按钮）

### 2.3 数据统计 ✅
- 月度收支总览卡片
- 支出类别占比饼图
- 近 7/30 天趋势折线图

### 2.4 预算管理 ✅
- 月度总预算设置
- 已用/剩余额度 + 进度条
- 超预算红色警告

### 2.5 导入导出 ✅
- CSV / Excel 导出
- CSV / Excel 导入（预览 + 字段映射 + 校验）
- JSON 完整备份与恢复

### 2.6 移动端体验（当前阶段）
- 底部 Tab 导航（≤768px）
- 响应式布局
- 安全区域适配
- 触摸友好交互
- 离线可用（PWA Service Worker）

## 3. 数据存储
- 浏览器端：sql.js (WebAssembly) + localStorage
- 完全离线，无需网络
- 支持 JSON 导入导出备份
