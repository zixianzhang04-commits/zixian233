# 开发日志说明

`dev-logs/` 用于记录每日开发完成事项与待办事项。

## 使用方式
1. 每天开始或结束开发时，运行 `node scripts/daily-log.js`。
2. 脚本会在 `dev-logs/` 下生成当天日志文件，文件名格式为 `YYYY-MM-DD.md`。
3. 请在当天日志中记录：
   - 完成事项
   - 待办事项
   - 重要问题或风险

## 日志内容示例

```md
# 2026-06-13 开发日志

## 完成事项
- 已完成 Electron 项目初始化
- 设计数据库初始结构

## 待办事项
- 实现用户登录注册功能
- 搭建基础记账 CRUD 页面

## 问题与备注
- 需要确认数据备份目录策略
```

## 注意
- 如果当天日志已存在，脚本不会覆盖原文件；请直接编辑现有文件。
- 日志文件应保持简洁明了，便于回顾和交接。
